# AKS74U Tishina Mod

A weapon mod for **S.T.A.L.K.E.R. Anomaly / GAMMA** adding a reworked AKS-74U ("Tishina") with new
HUD animations, sounds, textures, attachments (PSO, 1P29, Kobra optics and a GP-25 grenade launcher),
and supporting scripts.

## Contents

The mod follows the standard `gamedata/` layout:

| Folder | Contents |
| --- | --- |
| `gamedata/configs/` | Weapon, addon and system `.ltx` configs plus localized text (`eng`/`rus`) |
| `gamedata/scripts/` | Lua scripts (jam animations, mag check, GL shot fix, loadout injection, cover tilt) |
| `gamedata/meshes/` | Weapon, HUD and animation models (`.ogf`, `.omf`) |
| `gamedata/textures/` | Weapon, scope, cartridge and UI textures (`.dds`, `.thm`, `.ktx2`) |
| `gamedata/sounds/` | Weapon sounds (`.ogg`) |
| `gamedata/anims/` | Camera-effect animations (`.anm`) |

## Installation

Copy the `gamedata/` folder into your Anomaly / GAMMA installation directory (or install via a mod
manager such as MO2 by pointing it at the repository root).
