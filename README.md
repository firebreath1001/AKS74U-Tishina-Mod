# AKS74U Tishina Mod

A weapon mod for **S.T.A.L.K.E.R. Anomaly** adding a reworked AKS-74U with a new Grenade launcher: "Tishina" with new
HUD animations, sounds, textures and supporting scripts.

## Installation

Install via MO2 since it is an FOMOD. the FOMOD will guide you through the installation process
